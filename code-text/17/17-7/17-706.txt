                                  DC REGULATIONS

FAILURE TO FILE A BIENNIAL REPORT (§ 17-706)

706.1   	If a domestic entity required to file a biennial report under § 702
fails to file a timely biennial report, the failure to timely file shall be
grounds for administrative dissolution of the domestic entity by the
Superintendent.



706.2   	If a foreign entity required to file a biennial report under § 702
fails to file a timely biennial report, the failure to timely file shall be
grounds for the termination of the foreign entity’s registration by the
Superintendent.









SOURCE: Final Rulemaking published at 60 DCR 13139 (September 20, 2013).
